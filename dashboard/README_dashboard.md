# Bluestock MF Analytics — Streamlit Dashboard

## How to Run

```bash
# From project root:
pip install streamlit
streamlit run dashboard/app.py
```

Opens at: http://localhost:8501

## Pages

| Page | Contents |
|------|----------|
| 🏠 Industry Overview | AUM by AMC, folio count, KPI cards |
| 📈 Fund Performance  | Risk-return scatter, NAV chart, scorecard table |
| 👥 Investor Analytics | State map, age group, SIP/Lumpsum split |
| 📊 SIP & Market Trends | SIP vs Nifty dual-axis, category heatmap |

## Filters (Sidebar)
- Fund House, Category, Plan (Page 2)
- State, Age Group, City Tier (Page 3)
